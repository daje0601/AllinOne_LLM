## 📚 소개

이 책은 최신 LLM(Large Language Model) 기술의 모든 것을 다룹니다. NLP의 역사적 배경부터 실전에서 바로 활용할 수 있는 다양한 튜닝 기법(Full-Fine Tuning, Lora Tuning, QLora Tuning)과 vLLM 서빙 방법까지, LLM 전문가가 되기 위한 모든 내용을 담았습니다.

## 📌 주요 내용

- **이론 & 역사**
  - NLP의 발전 과정
  - 신경망과 역전파 알고리즘
  - GPT 모델의 학습 원리

- **실전 튜닝 기법**
  - Full Fine-tuning 구현
  - LoRA Tuning 최적화
  - QLoRA Tuning 테크닉

- **모델 서빙**
  - Production 환경 배포
  - 성능 최적화
  - 모니터링
